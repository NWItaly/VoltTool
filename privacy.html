👤 Titolare e contatti

VoltTool è un progetto open source dedicato a piccoli strumenti per la vita in elettrico, pubblicato su GitHub all'indirizzo: nwitaly.github.io/VoltTool.

Il titolare del trattamento è nwitaly, maintainer del repository GitHub. Per qualsiasi richiesta o segnalazione relativa alla privacy puoi aprire una issue o utilizzare i contatti indicati direttamente nel repository.

📋 Dati personali trattati

VoltTool non richiede registrazione, non prevede aree riservate e non utilizza moduli di contatto, newsletter o sistemi di login.

Non vengono raccolti, inviati o memorizzati deliberatamente dati personali degli utenti (nome, cognome, indirizzo email, ecc.) tramite questo sito.

💾 Salvataggio locale (localStorage)

Gli strumenti di VoltTool salvano le preferenze e i valori degli slider nel localStorage del browser, esclusivamente per ripristinare lo stato dell'interfaccia alla visita successiva.

Questi dati non lasciano mai il dispositivo dell'utente: non vengono trasmessi a nessun server remoto né letti da terze parti. Il localStorage non è un cookie e non viene inviato nelle richieste HTTP.

💡 Per cancellare questi dati è sufficiente svuotare la cache/storage del browser per questo sito, oppure utilizzare la modalità di navigazione privata.

🖥 Log tecnici e hosting

Il sito è ospitato tramite GitHub Pages. I log tecnici generati dall'infrastruttura di hosting (indirizzo IP, user agent, richieste HTTP) sono gestiti direttamente da GitHub secondo le proprie policy.

Per maggiori dettagli consulta l' informativa sulla privacy di GitHub.

🍪 Cookie e strumenti di tracciamento

Questo sito non utilizza cookie di profilazione e non impiega strumenti di analisi comportamentale o marketing (Google Analytics, Facebook Pixel e simili).

Non vengono impiegati cookie di terze parti. L'unico storage utilizzato è il localStorage descritto nella sezione precedente, strettamente necessario al funzionamento dell'interfaccia.

🌐 Risorse esterne

I font tipografici utilizzati dal sito (DM Mono, Syne) sono forniti localmente (self-hosted) e la libreria grafica Chart.js è inclusa direttamente nel repository. Non vengono effettuate chiamate a CDN o API esterne per il caricamento di questi asset.

La consultazione delle pagine non comporta pertanto la trasmissione dell'indirizzo IP dell'utente a servizi di terze parti per il semplice funzionamento dell'interfaccia.

🔧 Dati inseriti negli strumenti

Tutti i calcoli di VoltTool avvengono esclusivamente nel browser. I valori inseriti (consumi, prezzi, km annui, ecc.) non vengono mai inviati a server remoti né memorizzati dal titolare del sito.

L'utente può cancellare i dati in qualsiasi momento svuotando il localStorage dal browser o semplicemente utilizzando la navigazione privata.

⚖️ Diritti dell'interessato (GDPR)

Poiché il sito non raccoglie né conserva dati personali tramite proprie funzionalità, l'esercizio dei diritti previsti dagli artt. 15–22 GDPR (accesso, rettifica, cancellazione, opposizione, portabilità) riguarderà principalmente i log tecnici gestiti da GitHub Pages.

Per richieste relative ai log di hosting fai riferimento alla privacy policy di GitHub. Per qualsiasi altra richiesta puoi contattare il maintainer tramite il repository.

📅 Aggiornamenti a questa informativa

Questa informativa può essere soggetta a modifiche per adeguamenti normativi o tecnici. La data di "ultimo aggiornamento" in cima alla pagina indica sempre la versione corrente. Le modifiche sostanziali saranno segnalate nel changelog del repository.

🔒 Privacy Policy

Last updated: March 2026
✓ No profiling cookies ✓ No tracking ✓ All calculations local ✓ No account required ✓ Open source

👤 Data controller & contact

VoltTool is an open source project providing small tools for everyday EV life, published on GitHub at: nwitaly.github.io/VoltTool.

The data controller is nwitaly, maintainer of the GitHub repository. For any privacy-related request or report you can open an issue or use the contact details provided directly in the repository.

📋 Personal data processed

VoltTool requires no registration, has no restricted areas, and does not use contact forms, newsletters, or login systems.

No personal data (name, surname, email address, etc.) is deliberately collected, transmitted, or stored by this site.

💾 Local storage (localStorage)

VoltTool's tools save slider preferences and values in the browser's localStorage, solely to restore the interface state on subsequent visits.

This data never leaves your device: it is never transmitted to any remote server or read by third parties. localStorage is not a cookie and is not sent in HTTP requests.

💡 To delete this data, simply clear the browser storage for this site, or use private/incognito browsing mode.

🖥 Technical logs & hosting

The site is hosted via GitHub Pages. Technical logs generated by the hosting infrastructure (IP address, user agent, HTTP requests) are managed directly by GitHub under their own policy.

For more details please refer to GitHub's Privacy Statement.

🍪 Cookies & tracking

This site does not use profiling cookies and does not employ behavioural analytics or marketing tools (Google Analytics, Facebook Pixel, or similar).

No third-party cookies are used. The only storage in use is the localStorage described in the previous section, strictly necessary for the interface to function.

🌐 External resources

The typefaces used by the site (DM Mono, Syne) are served locally (self-hosted) and the Chart.js graphics library is bundled directly in the repository. No calls are made to external CDNs or APIs to load these assets.

Browsing the site therefore does not result in the user's IP address being transmitted to any third-party service merely for the interface to work.

🔧 Data entered in the tools

All VoltTool calculations run exclusively in the browser. Values entered (consumption figures, prices, annual km, etc.) are never sent to remote servers or stored by the site owner.

You can delete this data at any time by clearing the browser localStorage for this site, or by using private browsing mode.

⚖️ Your rights (GDPR)

Because the site does not collect or store personal data through its own features, the exercise of rights under Arts. 15–22 GDPR (access, rectification, erasure, objection, portability) will primarily concern the technical logs managed by GitHub Pages.

For requests relating to hosting logs please refer to GitHub's Privacy Statement. For any other request you can contact the maintainer via the repository.

📅 Updates to this policy

This policy may be updated to reflect regulatory or technical changes to the site. The "last updated" date at the top of the page always reflects the current version. Substantial changes will be noted in the repository changelog.
